RippleDrawable的简单使用

RippleDrawable 就是一个自定义触摸反馈动画。是在 Android 5.0 时推出的动画效果。为了兼容低版本，可以在将 元素定义在 drawable-v21 目录下；

首先定义一个 StateListDrawable 资源；

button_default.xml :

需要注意 时有序的。Android在遍历烈表时，会返回第一个状态完全匹配的 Drawable 。所以将常规状态放在烈表的最底部。

使用一些动画触摸反馈修饰按钮示例。使用 元素定义 XML (在 drawable-v21目录下)，可用于构造 RippleDrawable；

background_button.xml :

增加尺寸的缩放效果（res/animator 目录下）；

button_press.xml :

虽然 XML 结构使用和前面的 button_default.xml 相同的 ，但是在 res/animator 目录下，表示动画器实例而非 Drawable 的状态列表集合。
