Android应用通知是否开启状态的检测

APP大部分都会集成推送功能,随着Android系统的不断优化升级 用户体验不断的提供。大家经常会收到很多自己不想收到的通知,此时我们可能会在设置里面把APP 设置页面把通知栏功能给关掉。此后APP就无法正常发送通知消息了。

但作为一名开发者,我们有必要在合适的时候知道自己的功能缺失 并善意的提醒用户如何开启。

下面是从stackoverflow.找到的解决方法：
Now you can check it, as said in this Google I/O 2016 video.

Use NotificationManagerCompat.areNotificationsEnabled(), from support library, to check if notifications are blocked on API 19+. The versions below API 19 will return true (notifications are enabled).
