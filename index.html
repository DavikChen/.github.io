Mac下启动本地SVN服务器操作实例

在PC端有很好的本地SVN服务器软件,那在Mac上我们应该如何搭建本地SVN 服务器呢。其实Mac本身就可以做到这一点。
Mac默认已经安装了svn，我们只需要进行配置并开启就可以了首先我们可以验证一下是否安装了svn

没啥问题的话 可以显示svn相关版本信息

SVN目录配置:
1.创建本地目录 sudo mkdir -p /Users/xxx/dirName
2.sudo svnadmin create /Users/xxx/dirName 此时切换到目录下可以看到svn相关的文件已经创建完成
SVN权限配置:
1.在仓库目录下的conf目录里面 有一个svnserve.conf文件 打开文件后如下图

修改以上位置，其中anon-access = read代表匿名访问的时候是只读的，若改为anon-access = none代表禁止匿名访问，需要帐号密码才能访问
修改同目录下的passwd文件
在文件尾部以userName= password的形式添加用户

配置authz文件

在[groups]下添加uesrs = davikchen标示创建了一个用户组，此用户组包含有davikchen用户
[/]
@users = rw 这两句标示给users用户组相应的权限
[/]表示授权的目录路径，这里是根目录，假如根目录下有一个目录叫做test,那么我们如果要编辑此目录的权限那么就要写成[test:/]
@uesr表示给用户组授权，如果要给某一个用户授权则不用写前面的@
r表示可读，w表示可写

启动svn服务器
在终端输入
svnserve -d -r /Users/xxx/dirName
注意这里是仓库的根目录

这样SVN服务器算配置完成了 。根据我们刚才配置的账号 我们可以通过 SVN工具Cornerstone 链接到本地仓库。

Android_shape的使用

在APP开发中 我们经常会用到xml来设置一个背景 而且有圆角 渐变 虚线等。一般UI设计师提供切图也可以 不过代码能实现的 我一般就不让UI设计师切图给我们了 只需要告诉研发人员 相关标注就OK了 。

下面是shape的一些属性使用例子：
shape用于设定形状，可以在selector，layout等里面使用，有6个子标签，各属性如下

填充：设置填充的颜色

间隔：设置四个方向上的间隔

大小：设置大小

圆角：同时设置五个属性，则Radius属性无效

android:Radius=”20dp” 设置四个角的半径

android:topLeftRadius=”20dp” 设置左上角的半径
android:topRightRadius=”20dp” 设置右上角的半径
android:bottomLeftRadius=”20dp” 设置右下角的半径
android:bottomRightRadius=”20dp” 设置左下角的半径

描边：dashWidth和dashGap属性，只要其中一个设置为0dp，则边框为实现边框

android:width=”20dp” 设置边边的宽度
android:color=”@android:color/black” 设置边边的颜色
android:dashWidth=”2dp” 设置虚线的宽度
android:dashGap=”20dp” 设置虚线的间隔宽度

渐变：当设置填充颜色后，无渐变效果。angle的值必须是45的倍数（包括0），仅在type=”linear”有效，不然会报错。android:useLevel 这个属性不知道有什么用。

这是基本的使用 后续在复习下更复杂的应用
